## Što paušalac mora napraviti na kraju godine?

Kraj fiskalne godine (31. prosinca) za paušalnog obrtnika znači:

1. **Predaja godišnjeg PO-SD obrasca** (do 28. veljače)
2. **Provjera plaćenih doprinosa** (mirovinsko i zdravstveno)
3. **Predaja DOH obrasca** (samo ako imate dodatne prihode)
4. **Arhiviranje dokumentacije** (6 godina)
5. **Planiranje za sljedeću godinu** (prelazak u sustav PDV, promjena oblika oporezivanja)

## Rok: 28. veljače 2025 za prethodnu godinu

**Ključni datumi:**

- **31. prosinac prethodne godine** - Završetak fiskalne godine
- **20. siječnja 2025** - Predaja PO-SD za Q4 prethodne godine
- **28. veljače 2025** - Predaja godišnjeg PO-SD za prethodnu godinu
- **28. veljače 2025** - Predaja DOH obrasca (ako je potreban)

## Korak 1: Predaja godišnjeg PO-SD obrasca

Godišnji PO-SD je **zbirni obrazac** koji sumira sve kvartalne PO-SD obrasce za godinu.

### Što trebate pripremiti?

- Sva 4 kvartalna PO-SD obrasca (Q1, Q2, Q3, Q4)
- Ukupne primitke za cijelu godinu
- Bankovne izvode za provjeru

### Primjer godišnjeg izračuna

Paušalac iz Zagreba s ukupnim primitcima **35.000 EUR** u prethodnoj godini:

| Stavka                       | Izračun        | Iznos                 |
| ---------------------------- | -------------- | --------------------- |
| Ukupni primici (godina)      | -              | 35.000 EUR            |
| Porezna osnovica (12%)       | 35.000 × 12%   | 4.200 EUR             |
| **Godišnji porez:**          |                |                       |
| - Porez na dohodak (20%)     | 4.200 × 20%    | 840 EUR               |
| - Prirez Zagreb (18%)        | 840 × 18%      | 151,20 EUR            |
| **Ukupan porez**             | 840 + 151,20   | **991,20 EUR**        |
|                              |                |                       |
| **Već plaćeno (kvartalno):** |                |                       |
| - Q1 doprinosi               |                | 230 EUR               |
| - Q2 doprinosi               |                | 310 EUR               |
| - Q3 doprinosi               |                | 290 EUR               |
| - Q4 doprinosi               |                | 350 EUR               |
| **Ukupno plaćeno**           |                | **1.180 EUR**         |
|                              |                |                       |
| **Razlika**                  | 1.180 - 991,20 | **Povrat 188,80 EUR** |

**Objašnjenje:** Kvartalni doprinosi su često viši od godišnjeg poreza, pa paušalci dobivaju povrat.

## Korak 2: Provjera plaćenih doprinosa

Prije godišnjeg obračuna provjerite jesu li svi kvartalni doprinosi plaćeni.

### Kako provjeriti stanje dugovanja?

1. **ePorezna → Pregled dugovanja**
   - Vidjet ćete sve dospjele, a neplaćene obveze
   - Ako postoje dugovanja, platite ih prije godišnjeg obračuna

2. **Kvartalni doprinosi:**
   - Mirovinsko osiguranje (20% osnovice)
   - Zdravstveno osiguranje (16,5% osnovice)

### Pozivi na broj za uplatu

| Doprinos               | Poziv na broj   | Primatelj                |
| ---------------------- | --------------- | ------------------------ |
| Mirovinsko osiguranje  | 68 XXXXXXXX 001 | RHMIO (OIB + oznaka 001) |
| Zdravstveno osiguranje | 68 XXXXXXXX 002 | HZZO (OIB + oznaka 002)  |

**Napomena:** Zamijenite XXXXXXXX s vašim OIB-om.

### Što ako niste platili doprinose?

- **Kamate:** Postoje kamate na kašnjenje (trenutno ~7% godišnje)
- **Blokada:** Nakon 60 dana dugovanja moguća je blokada računa
- **Rješenje:** Platite dospjele doprinose prije godišnjeg obračuna

## Korak 3: DOH obrazac (ako je potreban)

### Kada paušalac MORA predati DOH?

DOH obrazac (godišnja prijava poreza na dohodak) obvezni ste predati ako:

1. **Imate dodatne prihode:**
   - Primitak od zakupa (najamnina)
   - Dividende (ako ste dioničar d.o.o.)
   - Kamate na štednju (preko 133 EUR godišnje)
   - Kapitalni dobitak (prodaja nekretnina, dionica)

2. **Promijenili ste oblik oporezivanja u toku godine:**
   - Prelazak s paušalnog na stvarne troškove
   - Prelazak s obrta na d.o.o.

3. **Zahtijevate povrat poreza:**
   - Izdaci za zdravstvo, obrazovanje, darovanje
   - Doplatka za djecu

### Kada paušalac NE mora predati DOH?

Ako ste **samo paušalac** bez dodatnih prihoda - **NE** trebate predati DOH. Godišnji PO-SD je dovoljan.

### Predaja DOH obrasca

- **Rok:** 28. veljače 2025 (za prethodnu godinu)
- **Način:** ePorezna → Godišnja prijava poreza na dohodak (DOH)

## Česte greške pri godišnjem obračunu

**1. Propuštanje roka**

- Kazna: 200-2.000 EUR
- Kamate na neplaćeni porez

**2. Zaboravljanje dodatnih prihoda**

- Primjer: Zarada od zakupa 5.000 EUR nije prijavljena → Utaja poreza
- Kazna: Značajna novčana kazna

**3. Nepodudaranje kvartalnih i godišnjeg PO-SD**

- Ukupni iznos godišnjeg PO-SD MORA biti jednak zbroju kvartalnih

**4. Neprovjereno dugovanje**

- Godišnji obračun neće biti prihvaćen ako postoji dospjelo dugovanje

## Nakon predaje godišnjeg obračuna

**1. Čekanje rješenja (do 30. travnja)**
Porezna uprava šalje rješenje o godišnjem porezu na dohodak:

- **Doplata:** Ako imate dodatne prihode ili propušteno plaćanje
- **Povrat:** Ako su kvartalni doprinosi bili veći od godišnjeg poreza

**2. Plaćanje ili povrat**

- **Doplata:** Rok 15 dana od primitka rješenja
- **Povrat:** Automatski povrat na račun unutar 30-60 dana

**3. Žalba (ako niste zadovoljni)**
Rok za žalbu: 15 dana od primitka rješenja

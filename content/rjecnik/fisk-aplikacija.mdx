## Što je FiskAplikacija?

**FiskAplikacija** je službeni web portal Porezne uprave za upravljanje **Fiskalizacijom 2.0:**

- Registracija informacijskog posrednika
- Testiranje e-računa
- Pregled izdanih računa
- Upravljanje korisnicima

**Pristup:** https://fiskaplikacija.porezna-uprava.hr/

## Registracija

**Prijava:**

- **NIAS** autentikacija (e-osobna, token, certifikat)
- Potreban OIB poduzetnika

**Prvi put (do 31.12.2025.):**

1. Prijava na FiskAplikaciju
2. Odabir informacijskog posrednika (MIKROeRAČUN ili privatni)
3. Potvrda odabira
4. Aktivacija

## Funkcije FiskAplikacija

**Za poduzetnike:**

- Odabir posrednika
- Pregled e-računa
- Test okruženje (rujan-prosinac 2025.)
- Upravljanje ovlašenjima

**Za posrednike:**

- Registracija kao Access Point
- Tehnička dokumentacija
- API pristup
- Testiranje

## Testna faza (1.9.-31.12.2025.)

**Besplatno testiranje:**

- Izdavanje testnih e-računa
- Provjera integracije
- Simulacija PEPPOL-a
- Bez posledica

**Pripremite se za 1.1.2026!**
